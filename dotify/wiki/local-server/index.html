Серверы

Как пользоваться когда основные сервера не работают?

Если основные сервера Dotify недоступны, вы можете использовать локальный сервер для работы с музыкальными платформами.


Настройка локального сервера:


Шаг 1: Вход в систему
1. В окне входа выберите "Войти как гость"


Шаг 2: Переход к настройкам
1. Зайдите в "Настройки"
2. Перейдите во вкладку "Сервисы"
3. Выберите первую вкладку с сервером


Шаг 3: Скачивание локального сервера
1. Скачайте локальный сервер по ссылке: https://dotify.fun/DotifyServer.exe
2. Если не качается - отключите антивирус
3. Если не запускается - тоже отключите антивирус
4. Переместите файл по пути: "C:\\Users\\вашпк\\AppData\\Roaming\\com.dotify.app\\bin"


Шаг 4: Запуск сервера
1. Запустите сервер (можно вручную через файл)
2. Дождитесь полной загрузки


Шаг 5: Настройка платформы
1. Зайдите во вкладку площадки, с которой будете слушать музыку (например, SoundCloud)
2. Поставьте "Проверка токена" на локальный сервер
3. Нажмите на любой из ID (для других площадок вводите соответствующие данные)


Шаг 6: Активация локального режима
1. Если видите успешное сообщение, в настройках режимов выполнения выберите "Все локально"


Шаг 7: Проверка работы
1. Проверьте, работает ли поиск
2. Если нет - включите любой VPN или Zapret (добавьте адреса платформ в list general)


Если не работают данные:
- Проверяйте токены (для ВК обязательно выбирайте токен ВК ЗВОНКИ)
- Копируйте либо всю ссылку, либо точный токен (после = до &)
- Попробуйте включить VPN, если платформа заблокирована в вашей стране


Поддержка:
Если ничего не помогает, напишите тикет:
- https://dsc.gg/dotify
- https://t.me/nodotify_bot


Важно: Если сервер не качается и не включается - отключите антивирус. Сервер распаковывает себя для работы, у него нет подписи, он использует интернет - вот причины жалоб антивирусов.
